# تفعيل وظائف الإدارة في الداشبورد

## نظرة عامة
تم تفعيل جميع وظائف الإدارة في الداشبورد لتشمل إدارة الكورسات والمدونات والمستخدمين مع واجهة مستخدم حديثة ومتجاوبة.

## الميزات الجديدة

### 1. إدارة الكورسات
- **إضافة كورس جديد**: إمكانية إضافة كورسات جديدة مع جميع التفاصيل
- **تعديل الكورسات**: تعديل بيانات الكورسات الموجودة
- **حذف الكورسات**: حذف الكورسات غير المرغوب فيها
- **عرض الكورسات**: عرض جميع الكورسات مع التفاصيل الكاملة

#### حقول الكورس:
- عنوان الكورس
- وصف الكورس
- اسم المدرب
- المدة (بالساعات)
- السعر
- التصنيف (برمجة، تطوير الويب، تطوير الهواتف، علوم البيانات، الأمن السيبراني، أخرى)
- الوسوم

### 2. إدارة المدونات
- **إضافة مدونة جديدة**: إمكانية إضافة مدونات جديدة
- **تعديل المدونات**: تعديل محتوى المدونات الموجودة
- **حذف المدونات**: حذف المدونات غير المرغوب فيها
- **عرض المدونات**: عرض جميع المدونات مع التفاصيل

#### حقول المدونة:
- عنوان المدونة
- محتوى المدونة
- التصنيف (عام، تقنية، برمجة، تطوير الويب، أمن سيبراني، ذكاء اصطناعي، أخرى)
- الوسوم

### 3. إدارة المستخدمين
- **إضافة مستخدم جديد**: إمكانية إضافة مستخدمين جدد
- **تعديل المستخدمين**: تعديل بيانات المستخدمين
- **حذف المستخدمين**: حذف المستخدمين
- **عرض المستخدمين**: عرض جميع المستخدمين مع التفاصيل

#### حقول المستخدم:
- الاسم الكامل
- اسم المستخدم
- البريد الإلكتروني
- رقم الهاتف
- كلمة المرور (للإضافة فقط)
- صلاحيات المدير

## API Endpoints الجديدة

### إدارة المستخدمين
- `GET /api/admin/users` - جلب جميع المستخدمين
- `POST /api/admin/users` - إضافة مستخدم جديد
- `PUT /api/admin/users/:userId` - تحديث مستخدم
- `DELETE /api/admin/users/:userId` - حذف مستخدم

### إدارة المدونات
- `GET /api/admin/blogs` - جلب جميع المدونات
- `POST /api/admin/blogs` - إضافة مدونة جديدة
- `PUT /api/admin/blogs/:blogId` - تحديث مدونة
- `DELETE /api/admin/blogs/:blogId` - حذف مدونة

### إدارة الكورسات
- `GET /api/admin/courses` - جلب جميع الكورسات
- `PUT /api/admin/courses/:courseId` - تحديث كورس
- `DELETE /api/admin/courses/:courseId` - حذف كورس

## الميزات التقنية

### 1. واجهة المستخدم
- **تصميم حديث**: واجهة مستخدم عصرية مع ألوان متدرجة
- **متجاوبة**: تعمل على جميع أحجام الشاشات
- **دعم الوضع المظلم**: تبديل سلس بين الوضع المظلم والفاتح
- **أيقونات Font Awesome**: أيقونات واضحة وجذابة

### 2. النماذج التفاعلية
- **نماذج منبثقة**: نماذج منبثقة لإضافة وتعديل البيانات
- **التحقق من البيانات**: التحقق من صحة البيانات قبل الإرسال
- **رسائل النجاح والخطأ**: رسائل واضحة للمستخدم
- **تحميل البيانات**: تحميل البيانات الموجودة للتعديل

### 3. الأمان
- **حماية بالتوكن**: جميع العمليات محمية بتوكن JWT
- **صلاحيات المدير**: الوظائف متاحة للمديرين فقط
- **التحقق من البيانات**: التحقق من صحة البيانات على الخادم

## كيفية الاستخدام

### 1. الوصول للداشبورد
1. تسجيل الدخول بحساب مدير
2. الانتقال إلى صفحة البروفايل
3. النقر على "لوحة التحكم" في القائمة الجانبية

### 2. إدارة الكورسات
1. النقر على "الكورسات" في القائمة الجانبية
2. النقر على "إضافة كورس" لإضافة كورس جديد
3. ملء البيانات المطلوبة
4. النقر على "إضافة" لحفظ الكورس

### 3. إدارة المدونات
1. النقر على "المدونات" في القائمة الجانبية
2. النقر على "إضافة مدونة" لإضافة مدونة جديدة
3. كتابة العنوان والمحتوى
4. اختيار التصنيف والوسوم
5. النقر على "إضافة" لحفظ المدونة

### 4. إدارة المستخدمين
1. النقر على "المستخدمين" في القائمة الجانبية
2. النقر على "إضافة مستخدم" لإضافة مستخدم جديد
3. ملء البيانات المطلوبة
4. اختيار صلاحيات المدير
5. النقر على "إضافة" لحفظ المستخدم
